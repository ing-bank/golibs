# Contributing to GoLibs

Thank you for your interest in contributing to GoLibs!

## How to Contribute

- **Bug Reports & Feature Requests:**
  - Please open an issue describing the bug or feature you would like to see.
- **Pull Requests:**
  - Fork the repository and create your branch from `main`.
  - Add tests for new features or bug fixes.
  - Ensure all tests pass (`go test ./...`).
  - Follow the existing code style and conventions.
  - Reference the related issue in your PR description if applicable.
- **Documentation:**
  - Improvements to documentation are welcome! Update README.md or package docstrings as needed.
